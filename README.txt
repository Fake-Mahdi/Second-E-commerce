E-Commerce Website Overview
This project showcases a fully functional e-commerce website built from the ground up using HTML, CSS, JavaScript, and Bootstrap 3. The site's architecture relies on JSON files that serve as the backbone for storing and managing product properties, enabling dynamic content rendering on the web pages.

Features
Dynamic Product Display: Products are dynamically loaded and displayed from JSON files, ensuring that the product data is easy to manage and update.
Responsive Design: Leveraged Bootstrap 3 to ensure that the website is responsive and provides a seamless user experience on devices of various sizes.
Interactive Elements: Includes interactive search bars and buttons, enhancing user engagement and ease of navigation.
Contact and Information Sections: Dedicated sections for contact information and further details about products and promotions enhance user interaction and accessibility.
Technologies Used
HTML: For structuring the web content.
CSS: For styling all the web components consistently.
JavaScript (main.js): For handling the website's interactivity and loading data from JSON files.
Bootstrap 3: For responsive design, making the website mobile-friendly and accessible on all devices.
JSON: For storing and managing the product data, making the site easily scalable and manageable.
Project Structure
index.html: The main entry file for the website, containing the structure and linking other resources.
style.css: Contains all the custom styles for the website.
main.js: Handles the dynamic loading of products and other interactive features.
products.json: A JSON file that contains data about the products available in the store.
